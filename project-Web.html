Statická HTML stránka

Tato statická stránka slouží jako můj osobní informační web, kde najdete důležité odkazy, informace o mně a mé projekty. Je postavena na šabloně Bootstrap a hostována na mém vlastním Ubuntu Linux serveru. Web je dostupný přes HTTPS a provozovaný pomocí NGINX.

Stránka je propojena s GitHub repozitářem PoweredByJecna/jirkachroust.cz. Díky nastavenému webhooku se po každém git pull na serveru stránka automaticky aktualizuje podle posledních změn z repozitáře.
